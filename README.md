# Frontend Developer Portfolio

A simple, clean personal portfolio website built with plain HTML and CSS. No frameworks, no dependencies.

## Preview

Ocean blue and white color scheme with a fully responsive layout across all screen sizes.

## Sections

- Hero — name, title, and call-to-action buttons
- About — bio, stats, and quick facts
- Skills — languages, frameworks, tools, and other tech
- Projects — project cards with links
- Contact — message form
- Footer — social links

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox)

## Getting Started

No build step required. Just open the file in a browser.

```bash
open index.html
```

## Customization

All personal content is in `index.html`. Update the following to make it yours:

- Name, title, and bio text
- About stats and quick facts
- Skills tags
- Project names, descriptions, and links
- Contact form action
- Social links in the footer

Colors are defined as CSS variables at the top of `style.css` and can be adjusted there.
